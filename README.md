# 🐾 PawWatch — Dog Behavior Monitoring System

Real-time dog emotion and behavior monitoring using computer vision.

**Tech:** MobileNetV2 · YOLOv8n · MediaPipe · Streamlit

**Emotions:** 😠 Angry · 😊 Happy · 😌 Relaxed · 😢 Sad

## Deploy on Streamlit Cloud
Set this in app Secrets (Settings → Secrets):
```
MODEL_URL = "your_hugging_face_URL"
```
